## 主要问题

1. **ES Module 兼容性**：
   - Jest 配置已经更新为使用 ES Module 语法 (`export default`)
   - 模块导入中仍然存在映射问题

2. **模块解析**：
   - 无法定位实际源文件：`Could not locate module ../../src/unminify/webcrack-unminifier.js`
   - 可能需要更新项目以适应测试环境，或者调整测试策略

3. **模拟模块**：
   - `cleye` 模拟不起作用，可能是由于 ES Module 的特性
   - 命令行选项测试需要重新设计
